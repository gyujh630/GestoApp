# [1] About the Project
- 제스처 기반 프레젠테이션 툴
- Vision Pro의 손동작 인식 및 조작 방식에 영감을 받아 제작했습니다.
  

## Features
- PDF 파일을 업로드하고, Guide에 있는 제스처를 사용해 화면을 제어할 수 있습니다.
- tap, double tap, pointer, zoom, fast zoom, tab control


## Technologies

- Node 18.17.1
- React 18.2.0
- Electron 28.2.0
- Vite 5.0.12
- TypeScript 5.3.3
- Pdfjs-dist 4.1.392
- zustand 4.5.2
- mediapipe/tasks-vision 0.10.13
